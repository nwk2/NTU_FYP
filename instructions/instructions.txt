# My Final Year Project


This repo contains my Final Year Project Product as well as relevant reports and instructions for my project for submission.

The contents are as follows:
- /bin/client - source code for client
- /bin/server - source code for server
- /instructions - instructions to set up environment/software
- /report - Final Report in PDF format

#### Installation:
Client
```sh
$ cd {path}/bin/client/fyp-frontend
$ npm install
$ npm start
```

Server
```sh
$ cd bin/server/pms
$ //active your virtualenv
$ pip install -r {path}/bin/server/pms/requirements.txt
$ python manage.py runserver
```
